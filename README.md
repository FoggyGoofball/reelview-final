# ReelView - Web Edition

Pure React streaming app for movies, TV shows, and anime.

?? **Live:** https://foggygoof ball.github.io/reelview-final/

## Branches

- **main** (you are here) - Web app
- **electron** - Desktop (Windows/Mac)
- **capacitor** - Mobile (Android/iOS)

## Quick Start

```bash
npm install
npm run dev        # Development server
npm run build      # Production build
```

## Features

- Browse movies, TV shows, anime by genre
- Continue watching functionality
- Watchlist management
- Download episodes/movies
- Chromecast support
- Watch history tracking
- Multiple video sources

## Technologies

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Capacitor (for mobile builds)
